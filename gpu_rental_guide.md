# Руководство по аренде GPU для обучения Kazakh NER

Поскольку обучение трансформера XLM-RoBERTa на CPU приводит к сбою Out-Of-Memory (OOM) и длится слишком долго, аренда облачного GPU — это самый быстрый, надежный и дешевый способ запустить проект (стоимость составит около **$0.20 – $0.50** за всё время работы, так как обучение займет менее 30 минут).

---

## 1. Рекомендуемые платформы для аренды GPU
1. **[RunPod.io](https://www.runpod.io/)** (Самый удобный интерфейс, быстрый запуск инстансов с Jupyter Notebook).
2. **[Vast.ai](https://vast.ai/)** (Самый дешевый шеринг видеокарт, большой выбор).
3. **[Paperspace Gradient](https://www.paperspace.com/)** (Отличные виртуальные машины, удобная среда).
4. **Google Colab Pro** (Простой запуск прямо в браузере, если есть подписка).

---

## 2. Параметры конфигурации сервера (Шаблон для создания инстанса)

При создании сервера (Pod / Instance) выбери следующие настройки:

### Выбор видеокарты (GPU)
* **Минимальная конфигурация**: **NVIDIA RTX 3090 / RTX 4090** (24 GB VRAM) или **NVIDIA A10G / L4** (24 GB VRAM).
  * *Пояснение*: Эти карты стоят около **$0.20 – $0.40/час** и справятся с обучением нашей модели за 15–20 минут.
  * *Альтернатива*: Tesla T4 или RTX 3060 (12 GB VRAM) тоже подойдут, но обучение займет около 40–50 минут.

### Ресурсы системы (System Specs)
* **Системная память (RAM)**: минимум **16 GB** (чтобы избежать повторных падений по OOM при подготовке токенов процессором).
* **Объем диска (Disk Space)**: **25–30 GB** (базовый образ Docker займет около 15 GB, плюс веса XLM-RoBERTa ~1.2 GB и чекпоинты обучения ~5 GB).

### Выбор Docker-образа (Template / Image)
Выбери стандартный PyTorch шаблон с поддержкой CUDA:
* **`pytorch/pytorch:2.2.1-cuda12.1-cudnn8-devel`**
* или **`runpod/pytorch:2.1.0-py3.10-cuda11.8.0-devel`** (на RunPod)

---

## 3. Пошаговый сценарий запуска после подключения к GPU

После того как сервер запустится, подключись к нему через Jupyter Lab (встроен в RunPod) или SSH.

### Шаг 1. Перенос файлов на сервер
Создай на сервере папку `project` и загрузи туда 4 файла:
* `main.py`
* `IOB2_train.txt`
* `IOB2_valid.txt`
* `IOB2_test.txt`

### Шаг 2. Выполнение команд установки в терминале сервера
Открой терминал внутри Jupyter Lab или подключись по SSH и выполни:

```bash
# Обновляем пакетный менеджер и устанавливаем зависимости
pip install --upgrade pip
pip install transformers datasets accelerate scikit-learn

# Запускаем обучение
python3 main.py
```

---

## 4. Ожидаемое время выполнения и результаты
* **Подготовка датасета**: ~1 минута.
* **Скачивание XLM-RoBERTa**: ~1 минута (скорость интернета в дата-центрах обычно от 1 Гбит/с).
* **Обучение (3 эпохи на RTX 3090/4090)**: ~15-20 минут.
* **Итоговый вывод на экран**: Скрипт выведет детальную таблицу метрик (Precision, Recall, F1 для каждого класса) и сохранит готовую модель в папку `./ner-kazakh-best`.

### Что скачать обратно на свой ноутбук:
Скачай созданную папку **`ner-kazakh-best`** (размер архива будет около 1.1 ГБ). Положи её в папку своего проекта на локальном ноутбуке и запускай API-сервер (`python3 api.py`). После скачивания модели сервер аренды GPU можно смело удалять (Terminate), чтобы остановить списание денег.
