# Инструкция для дообучения модели (Kazakh NER Pipeline)

Этот документ содержит руководство по запуску полного цикла обучения модели Named Entity Recognition (NER) для казахского языка на базе предобученного трансформера **XLM-RoBERTa**. 

Пожалуйста, передай эту инструкцию другу вместе с необходимыми файлами.

---

## 1. Какие файлы нужно переслать другу

Сформируй архив (например, `ner_project.zip`), содержащий следующие файлы из твоей директории `industrial_practice`:

1. `main.py` — основной скрипт обучения.
2. `IOB2_train.txt` — обучающий датасет.
3. `IOB2_valid.txt` — валидационный датасет.
4. `IOB2_test.txt` — тестовый датасет.

*Примечание: Презентацию и `api.py` другу отправлять не нужно.*

---

## 2. Системные требования к машине друга
- Наличие дискретной видеокарты NVIDIA (желательно с объемом VRAM от 8 ГБ: RTX 3060/4060/3070/3080/4090 и т.д.).
- Операционная система: Ubuntu/Debian, Fedora, Arch Linux или Windows (с установленной WSL2).
- Установленный Python 3.8 - 3.11.

---

## 3. Инструкция по запуску (для твоего друга)

### Шаг 1. Подготовка окружения
Открой терминал в распакованной папке с файлами и установи необходимые зависимости:

```bash
# Обновление pip
python3 -m pip install --upgrade pip

# Установка PyTorch с поддержкой CUDA (для GPU)
# Официальная команда для установки актуальной версии PyTorch:
pip3 install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu118

# Установка библиотек экосистемы Hugging Face и метрик
pip3 install transformers datasets accelerate scikit-learn
```

### Шаг 2. Запуск обучения
Запусти скрипт обучения следующей командой:

```bash
python3 main.py
```

### Что произойдет во время работы скрипта:
1. Загрузятся датасеты KazNERD (размер train-выборки снят с лимитов и обучается на полном объеме).
2. Начнется автоматическое скачивание базовой модели `xlm-roberta-base` (~1.12 GB) с Hugging Face Hub.
3. Запустится процесс **Fine-Tuning** на 3 эпохи. С поддержкой GPU (`fp16`) обучение займет примерно 15–40 минут в зависимости от видеокарты.
4. После окончания обучения на консоль выведутся метрики точности на тестовом датасете (Precision, Recall, F1-Score) и детальный отчет по классам.
5. Готовая дообученная модель сохранится в папку `./ner-kazakh-best` внутри рабочей директории.

---

## 4. Что друг должен вернуть тебе обратно

После успешного выполнения скрипта друг должен заархивировать и прислать тебе созданную папку:

📁 **`ner-kazakh-best/`**

Внутри неё будут лежать файлы модели:
- `model.safetensors` (или `pytorch_model.bin`) — обученные веса.
- `config.json` — конфигурация.
- `tokenizer.json`, `sentencepiece.bpe.model` и др. — файлы токенизатора.
